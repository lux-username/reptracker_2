## Timeliness review (prompted by the user)

Assessed how the free tier affects "advance notice in time to act." Conclusion:
negligible for reference data + long-lead hearings; the real dent is the ≤~24h
staleness on **short-notice** markups/hearings (warm path is index-only, refreshed
daily). The Congress.gov 5k/hr rate limit is *not* the blocker (~500 calls/run);
the true ceiling is Congress.gov's own publish latency, which is unmeasured.

## Decisions (rationale → decisions.md)

- Convergent cursor + always-swept head + single index blob, over chunk-by-chamber
  routes / external scheduler / monolithic pass.
- Warm path reads index-only (no per-request live head-check) — simplicity over
  immediate freshness; this tradeoff is exactly what created the #23 gap.
